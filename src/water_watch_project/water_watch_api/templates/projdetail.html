Project Details:


Environmental Sensor Station Simulation and Analysis


The central idea of this project is to simulate Water Watch stations in California, U.S. which keeps track of the Physical and Inorganic measurements of water in specified location.


Features: Implementation of various APIs, Dashboard, Map-based Web application for users, Data retrieved from REST web services provided by USGS.
